Ticket Selling Platform

In this project, I developed a ticket selling platform for organizing and managing events. The platform allows users to browse different events, view event details, and purchase tickets online. It includes features such as event categorization, ticket pricing, and secure payment processing. The goal was to create a user-friendly platform that simplifies the process of buying event tickets and enhances the overall event experience for users.

Technologies used

  • HTML & CSS
  • JavaScript
  • Bootstrap
  • Django
  • Python
