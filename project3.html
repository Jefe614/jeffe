Weather App

This project is a simple weather application that provides users with current weather conditions and forecasts for their location. Users can enter their location or allow the app to access their location automatically. The app displays information such as temperature, humidity, wind speed, and weather conditions. It also allows users to switch between different units of measurement (e.g., Celsius and Fahrenheit) and view weather forecasts for multiple days.

Technologies used

  • HTML & CSS
  • JavaScript
  • Bootstrap
