GYM management system

The GYM management system is a web application designed to streamline the management of a fitness center or gym. It includes features such as member management, class scheduling, billing, and reporting. The goal is to provide gym owners and managers with a comprehensive solution to efficiently manage their operations and improve member satisfaction.

Technologies used

  • HTML & CSS
  • JavaScript
  • Django
  • Python
  • PostgreSQL
